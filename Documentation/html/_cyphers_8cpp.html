Function: CaesarEncrypt() Parameters: offst -> number of times to rotate clockwise source -> string to encrypt result -> string to decrypt Description: Encryption function that uses Caesar cypher. This kind of cypher rotates the alphabet a designated number of letters. If the displacemente is 3, 'A' becomes 'D', 'B' becomes 'E' and so on. It only works with letters.

The remaining symbols are unaffected
